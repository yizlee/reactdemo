---
 这是一个类淘票移动端web应用
---

技术栈：react  /  react-created-dom  /  react-router <br>
功能介绍： <br>
        首页    <br>
            1，城市浮层；<br>
            2，导图轮播；<br>
            3，点击电影列表进入详情页；<br>
            4，底部用户切换栏；<br>
        电影详情页<br>
            1，评分组件；<br>
            2，评论筛选；<br>
            3. 多行文本折叠显示方案；<br>
         购票页面<br>
            1，选座交互<br> 
 ### 如何开始
 ##### install dependencies
npm install

##### serve with hot reload at localhost
npm run dev
